/**
 * http://www.geeksforgeeks.org/dynamic-programming-subset-sum-problem/
 * 
 * Given a set of non-negative integers, and a value sum, determine if 
 * there is a subset of the given set with sum equal to given sum.
 */

 Solution:
 =========
 Can have iterative or recursive solution.
 It is a NP-compelte problem.
 iterative solution has a pseudo-polynomial time complexity
 space complexity = O(n*sum)
