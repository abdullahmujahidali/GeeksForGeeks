 * Given an array A[] and a number x, check for pair in A[] with sum as x.
 * Write a C program that, given an array A[] of n numbers and another number x, 
 * determines whether or not there exist two elements in S whose sum is exactly x. 
 */

 Solution:
 ---------
 Implemented using hash map.
 Time Complexity = O(n)
 Space Complexity = O(n)
 can also be implemented by sorting first and then iterating.
 Time Complexity = O(nlogn)
 Space Complexity = depends on sorting mechanism.
